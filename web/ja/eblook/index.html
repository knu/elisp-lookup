eblook

eblook は、EB ライブラリを用いた対話型の電子辞書検索コマンドです。 簡単な設定とコマンドにより CD-ROM 書籍が利用出来ます。

  • ダウンロード
  • インストール上の注意
  • Windows 用バイナリ
  • OS/2 用バイナリ

ダウンロード

現在の最新バージョンは 1.3 です。一般的な UNIX 互換システムで動作します。

  • eblook-1.3.tar.gz (120KB)

eblook をコンパイルするには、あらかじめ EB ライブラリ のバージョン 2.0 以降をインストールしておく必要があります。 2000/7/14 時点での最新版は 2.3.8 となっています。

  • eb-2.3.8.tar.gz (680KB)

インストール上の注意

RedHat 6.x 等のシステムでは、/usr/local/lib にライブラリのパスが 通ってないために、そのまま configure; make install したのではリンクに 失敗してしまいます。/etc/ld.so.conf に次の行を追加し、su して ldconfig を 実行して下さい。

/usr/local/lib

MS Windows 用バイナリ

以下のものは、eblook 及び関連ツールの Windows 用バイナリです。

  • eblook.exe (88KB)
    eblook-1.3 のバイナリです。このままで使えます。
  • eblook13cyg.zip (260KB)
    eblook-1.3 及び eb-2.3.6 に含まれるツールをまとめたものです。 実行するのに Cygwin B20.1 の cygwin1.dll が必要です。
  • eblook13nocyg.zip (261KB)
    eblook-1.3 及び eb-2.3.6 に含まれるツールをまとめたものです。 実行するのに Cygwin 等は必要ありません。
