“Kojien” Japanese Language Dictionary

  • [Home Page]

“Lookup” can handle EBG/EBXA/EPWING (up to V6) dictionaries with `eblook' and EB libraries. When used with EBXA dictionaries, Kanji characters in the query can automatically be converted to Hiragana so that the dictionaries can handle the queries. Furthermore, when used with EPWING dictionaries, old Kanji characters （旧字） in the query can automatically be converted to new Kanji characters （新字）.
